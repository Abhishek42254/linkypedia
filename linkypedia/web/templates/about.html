About Linkypedia

The idea behind linkypedia is to help organizations that are putting content online to see how wikipedians are citing and annotating their stuff. Many libraries, museums and archives have embraced the new publishing medium of the Web by digitizing items and putting them on the Web. Traditionally these items have been organized and annotated by catalogers, archivists and curators around the world.

linkypedia aims to help reveal the connections between these digital curation communities. To let cultural institutions get a handle on the rich metadata and contextual information found in wikipedia...and to help wikipedians find more the rich seams of historic

If you are interested in how it all works linkypedia is a Python/Django application that uses the External Links Search built into Wikipedia to get a list of wikipedia pages and the relevant links. The project also uses the lovely Wikipedia API to fetch relevant details about the wikipedia page. The source code is available on GitHub.
